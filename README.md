# ARKAN SETTLEMENTS

منصة صرافة وتسوية دولية — موريتانيا · أنغولا · الصين.
موقع ثابت (HTML/CSS/JS) بلا أي تكلفة شهرية.

**الموقع الحي:** https://arkanrates.com

## الصفحات
| الملف | الوصف |
|---|---|
| `index.html` | الصفحة الرئيسية (حاسبة، أسعار، محركات، خطط، طلب) |
| `rates.html` | بوابة أسعار العملاء (ثلاث شرائح) |
| `request.html` | تحويلات الشركات + قراءة الفواتير OCR |
| `trading.html` | أنظمة التداول ARKAN AI |
| `settlement.html` | لوحة التسوية الداخلية (محمية برمز) |

## التكاملات — كلها مجانية
كل الخدمات تُفعّل من ملف واحد: **`config.js`**.
أي حقل يُترك فارغًا = الخدمة معطّلة بأمان دون أن يتأثر الموقع.

| الخدمة | ما تحتاجه | الحد المجاني |
|---|---|---|
| Google Analytics 4 | معرّف `G-XXXXXXX` | غير محدود |
| Tawk.to (دردشة حية) | معرّف الـ widget | دائم |
| OneSignal (إشعارات) | App ID | 10,000 مشترك |
| EmailJS (بريد الطلبات) | Public Key + Service + Template | 200 بريد/شهر |
| Firebase (حسابات + قاعدة بيانات) | إعدادات تطبيق الويب | 50k مستخدم · 50k قراءة/يوم |
| CoinGecko (أسعار حية) | **لا شيء — يعمل الآن** | 10-30 طلب/دقيقة |

### تفعيل خدمة
1. افتح `config.js`
2. الصق المفتاح في حقله
3. احفظ وارفع — تعمل فورًا

## PWA
- `site-manifest.json` + `sw.js` (Service Worker)
- يعمل دون اتصال، وقابل للتثبيت على الشاشة الرئيسية

## النشر
GitHub Pages تلقائيًا عند كل `push` على `main` عبر `.github/workflows/deploy.yml`.

## اللغات
عربي · فرنسي · برتغالي — كشف تلقائي من لغة الجهاز (`i18n.js`)، ومبدّل يدوي أسفل الشاشة.
